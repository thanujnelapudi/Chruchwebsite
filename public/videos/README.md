# Hero Video Background

Download the cloud sky timelapse video from Pexels and save it here as `hero-clouds.mp4`:

**Download URL:** https://www.pexels.com/video/time-lapse-video-of-clouds-4787839/

1. Visit the link above
2. Click the **Download** button
3. Select **Full HD (1920×1080)** or **HD (1280×720)**
4. Save the file as `hero-clouds.mp4` in this folder

The hero section will automatically use this video as the background.
If the video file is missing, the hero falls back to the static poster image.
